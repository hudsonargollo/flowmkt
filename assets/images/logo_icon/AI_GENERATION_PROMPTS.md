# AI Image Generation Prompts for FlowMkt Logos

This document provides ready-to-use prompts for generating FlowMkt brand assets using AI image generators like DALL-E, Midjourney, Stable Diffusion, or other tools.

## Brand Context

**FlowMkt** is a marketing automation platform focused on WhatsApp messaging and flow-based communication. The brand should convey:
- Automation and efficiency
- Flow and connectivity
- Modern technology
- Professional and trustworthy
- Brazilian market focus

## Post-Generation Instructions

### After Generating with AI Tools:

1. **Download the images** in the highest quality available

2. **Resize to exact specifications** using image editing tools:
   - **Photoshop**: Image > Image Size
   - **GIMP**: Image > Scale Image
   - **Online**: Use TinyPNG or Squoosh.app

3. **Ensure transparency**:
   - Save as PNG-24 with alpha channel
   - Remove any white/colored backgrounds
   - Check transparency in image editor

4. **Optimize file size**:
   - Use TinyPNG (https://tinypng.com)
   - Use Squoosh (https://squoosh.app)
   - Target: under 50KB for logos, under 500KB for backgrounds

5. **Test at different sizes**:
   - View logos at 100%, 50%, and 25% size
   - Ensure favicon is clear at 16x16px
   - Check contrast on both light and dark backgrounds
